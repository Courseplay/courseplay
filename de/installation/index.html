Installation

Installation der veröffentlichten Version

Kopiere das Archiv FS17_CoursePlay5.zip in deinen Mods-Ordner. Das war's!

Der Mods-Ordner befindet sich:

  • Windows (7 und höher):
  • C:\Users\[USER NAME]\My Documents\MyGames\FarmingSimulator2017\mods
  • Mac OS X (Download/DVD/Steam-Version):
  • /Users/[USER NAME]/Library/Application Support/FarmingSimulator2017
  • Mac OS X (X 10.7 und höher):
  • /Users/[USER NAME]/Library/Containers/com.focus-home.farmingsim2017/Data/Library/Application Support/FarmingSimulator2017

Installation der Entwickler-Version

First be sure you have the latest patch: Landwirtschafts-Simulator 17 Update

Get the latest developer version here: Developer Version

Entpacke die heruntergeladene Zip-Datei in einen Ordner deiner Wahl.
Hier findest du nun einen neuen Ordner namens courseplay-master.
Select all Files and Folders inside the courseplay-master folder and Archive the selected to a new zip file, named FS17_CoursePlay5.zip.
Dann installiere diese normal wie oben beschrieben.
