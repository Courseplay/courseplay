Changelog

Änderungen in Version v5.01 (FS17)

  • Fix wrong keybinding description.
  • Fix nil ridge markers error.
  • Mode 8: Manure transporter is renamed to Liquid product transport

  • Improved Grain Transport Mode

    • Vehicles at a silo should now center the trailers underneath the unloading point, no matter how far ahead the first waypoint is.
  • Allow reversing with tools in "Transport Mode".
  • Disable auto field scanning on field-less maps. (Typically maps that are made for forestry only)
  • Enabling small 0.1 steps when workWidth is over over 10.0, by holding down left Alt key.
    We now have three steps when over 10.0: 0.5 and 2.0 for fast setting and while holding left Alt even 0.1 for fine-tuning.

  • Mode 6: Fieldwork

    • Combine in this mode can now go unload like in mode 7 if there is an unload route attached to the course.

  • Mode 7: Combine self unloading

    • Combine can now create heaps as well.
    • This is done by adding 2 waiting point instead of 1 (which is used to unload into an trailer). One for the heap start and one for the end of the heap.
    • The combine will adjust the unload speed by it self while doing that.
    • This is also implemented into "Mode 6: Fieldwork" for combines.

  • Mode 9: Shovel Fill and Empty

    • This is now adapted to the new bunker type.

  • Lights

    • Now switch on an off based on it's situation. When working on the field all lights are turned on. When driving on an recorded route or unload/refill route, it will use normal street light without the work lights.

  • CP Hud Scaling.

    • CP hud scales based on the UI Scale setting.
    • Hud scale can also be set in the settings xml now.
  • Added new keybinding to toggle "stop at last point or next trigger" (Have no default keybinding, so needs to be set)

  • Reversed Driving Support

    • Support reversed driving for vehicles like the Valtra T Series.
  • Added new unload point while recording an course.
  • Implemented water delivery in mode8
  • Added always refill fuel feature
  • Added work width button to mode 9 settings
  • Added possibility to save the position of the overloaders pipe

  • New Turn System: Stage 1

    • This is stage 1, which means that it only effects lane turns and not the headlands, which will be implemented in stage 2.
    • The new turn system works for Seeding/Fertilize and Fieldwork mode.
    • Turn Types: Wide Turn, Wide Turn With Avoidance (Only used in very special cases), Ohm Turn, Questionmark Turn, Forward 3 Point Turn and Reverse 3 Point Turn.

  • New Save System

    • Each course is now saved in there own xml file inside CoursePlay_Courses\(mapId)\courseStorageXXXX.xml where XXXX is represented by an number.
    • There is a CoursePlay_Courses\(mapId)\courseManager.xml file there as well to control what course storage file is for which course. This file also controls folders.
    • All courses are saved outside the savegameX folder and will be available for each savegame that uses the same map, so no need to rerecord any courses for the same map once you starting an new savegame.
    • This also improves the loading time and fixes the problem with xml files getting to big to be saved.

  • New BunkerSilo Mode

    • New mode that allows using a shield to level and compacting the new bunkersilo system.
    • Tractors with an shield will clear out of the silo once it registers an incoming vehicle that want's to unload in the silo.
    • Vehicles that are coming in to unload will wait until the tractor with an shield have cleared out of the silo.

Änderungen in Version v4.01 (FS15)

  • Verbesserte Erkennung von angehängten Dreschern/Häckslern
  • Drescher-selbst-abtanken: der Drescher setzt ein wenig zurück, bevor er zum Abladepunkt fährt, um nicht direkt durch die Frucht zu fahren.

  • Vorwärts-Abladen in Silage-Silos

    • Es wird jetzt über das gesamte Silo verteilt in einem Schritt abgeladen, anstatt per Drei-Sektionen-Aufteilung.
    • Die Ablade-Fahrgeschwindigkeit wird automatisch berechnet und gesetzt.
  • Mitteilung wenn ein Gerät im Dünge/Sä- und Feldarbeitsmodus aufgefüllt/entleert werden muss
  • Debug-Levels sind jetzt Debug-Channels, die spezifische Debug-Möglichkeiten stark verbessern.
  • Der Geräte-Versatz kann nun nur in den Überlade-, Dünger-/Sä-, Feldarbeits und Drescher-selbst-abtanken-Modi benutzt werden.
  • Geschwindigkeiten werden nun korrekt für jede Sprache angezeigt.

  • Neues InputBinding-System

    • Maustasten können nun im Optionsmenü des Spiels geändert werden.
    • Deutlich bessere Kompatibilität mit anderen Mods die evtl. dieselben Knöpfe benutzen

  • Neues Überlader-System

    • Überladewagen nutzen nun nicht mehr ihre eigenen Specializations sondern werden direkt von Courseplay gesteuert.
    • Die meisten Überladewagen laden jetzt erst direkt am Überladepunkt ab, und nicht schon wenn sie noch in Bewegung sind.
    • Geräteversatz kann nun mit dem Überladekurs benutzt werden, was es ermöglicht verschiedenste Überladewagentypen auf dem selben Kurs zu benutzen.

  • Versätze

    • Bahn- und Geräteversätze sind nun separat, was eine bessere und genauere Positionierung ermöglicht.
    • Pflug-Versätze werden am Ende der Bahn umgekehrt, so dass das Gerät in der nächsten Bahn die korrekte Positionierung bekommt.
    • Drescher-selbst-abtanken: Geräteversatz kann nun mit dem Abladekurs benutzt werden, was es ermöglicht verschiedenste Drescher auf dem selben Kurs zu benutzen.

  • Automatischer und manueller Feldscanner

    • Courseplay kann die Felder der Map scannen, um Feldgrenzpfade zu generieren – Man muss nicht mehr um das Feld herumfahren.
    • In einem bestehenden Savegame ist diese Option standardmäßig aktiviert. Startet man ein neues Savegame erscheint ein Einstellungsdialog wo man die Option aktivieren/deaktivieren kann. Man kann den Scanner auch in der courseplay.xml Datei aktivieren/deaktivieren.
    • Zusätzlich können einzelne Felder manuell gescannt (wenn das Fahrzeug auf dem Feld steht) und gespeichert werden. Diese Felder werden in der courseplayFields.xml Datei gespeichert und beim nächsten Start wieder geladen.
    • Danke an upsidedown für die Initiierung und Hilfe.

  • Fahrer-Gehälter

    • Fahrern werden nun Gehälter bezahlt:
      • einfach: € 500 / Stunde
      • normal: € 1'000 / Stunde
      • schwer: € 1'500 / Stunde
    • Wird ein anderes Währungssystem als Euro benutzt, werden die Gehälter entsprechend angepasst.
    • In einem bestehenden Savegame ist diese Option standardmäßig aktiviert. Startet man ein neues Savegame erscheint ein Einstellungsdialog wo man die Option aktivieren/deaktivieren kann. Die Option kann auch in der courseplay.xml Datei aktiviert/deaktiviert werden.
  • Aktive CP-Fahrer werden auf der Ingame-Karte angezeigt. Diese Option kann in der courseplay.xml Datei aktiviert/deaktiviert werden. Zusätzlich kann eingestellt werden, ob der Fahrzeugname und der aktuelle Kursname angezeigt werden sollen.

  • Rückwärts abladen in Silage-Silos

    • Abladen in Silage-Silos kann nun auch per Rückwärtsfahren getätigt werden.
    • Das Abladen wird vom hinteren Ende des Silos begonnen.
    • Der Kurs muss ohne einen Hänger aufgezeichnet werden (als wäre der Traktor der Hänger). Fahre rückwärts bis zum hinteren Ende des Silos und von da aus wieder vorwärts.
    • Das Abladen beginnt ein paar Sektionen vor dem Ende – dadurch wird eine kleine Rampe erzeugt die besseres Handling ermöglicht.
  • Die Einstellung für die automatischen Spurreißer wird nun in der vehicles.xml Datei gespeichert und daraus geladen.
  • Schaufelpositionen werden nun in der vehicles.xml Datei gespeichert und wieder daraus geladen.
  • Die Courseplay-Version wird nun in der Hud angezeigt, und kann damit einfach und schnell an die Entwickler gemeldet werden wenn benötigt.

  • Einstellbare Hud-Position

    Die Hud-Position kann nun einfach in der courseplay.xml Datei geändert werden.
    (<courseplayHud posX="0.433" posY="0.002" />)

  • Mausrad-Benutzung

    Zahleneinstellung und Listen können nun auch mit dem Mausrad benutzt werden (wenn die Maus über dem entsprechenden Eintrag ist).
    Hinweis: während der Mauscursor aktiv ist, wird die Zoom-Funktion des Fahrzeugs deaktiviert.

  • Rückwärtsfahren mit Hängern

    • Hänger können nun um Kurven gefahren werden.
    • Nur ein (1) Hänger wird unterstützt, nicht mehrere gleichzeitig.
    • Der Traktor wird den Hänger immer in Richtung des nächsten Wegpunktes hinter dem Hänger ausrichten, daher sollte rechts und links genügend Platz zum rangieren sein.
    • Der Kurs sollte ohne Anhänger aufgezeichnet werden, um nicht die Lenkbewegungen des Gegensteuerns sondern nur den reinen Kurs aufzuzeichnen.
    • Beachtet aber die Anhängerlänge am Ende des Kurses. Der Radius bei Kurven sollte nicht zu eng sein.
    • Beim Rückwärtsfahren wird der Abladetrigger hinter dem Hänger gesucht.
  • Beim Annähern an Auffülltrigger (z.B. Saat, Benzin, Gülle …) wird das Fahrzeug langsamer um hartes Bremsen zu vermeiden.

  • Komplett neues Kurs-Management-System

    • Ordner, baby! Kurse (und Ordner) könnnen in andere Ordner verschoben werden um eine saubere Liste zu bekommen.
    • Durchsuchbare Kursliste: benutze einen Suchfilter um schnell den gewollten Kurs oder Ordner zu finden.
    • Schnelleres XML-Speichern und -Laden
    • Stark verbessertes Sortieren von Kursen
    • Danke an horoman, der das System schrieb und eingepflegt hat

  • Neue Kurs-Speichern/-Suchen GUI

    • Neues, sauberes Interface für bessere Sichtbarkeit und Benutzbarkeit
    • UTF8 wird nun in Kurs- und Ordnernamen unterstützt (ä, ö, ü, ß, é, ñ, ý etc.)
    • Maximallänge für Namen auf 25 Buchstaben erhöht

  • Komplett neues Wegpunkt-Zeichen-System

    • Extrem verbesserte Performance: unbenutzte Zeichen werden in einen Cache verschoben.
    • Zeichen werden endlich richtig gelöscht, was den Arbeitsspeicher entlastet.
    • Neue Low-Poly-Zeichen mit neuen Texturen

  • Abladen auf dem Feld

    Ballenlader können nun automatisch auf dem Feld abladen wenn kein Abladekurs angehängt wurde.

  • Traffic-Erkennungs-Raycast

    • Frühere Erkennung von Traffic-Fahrzeugen vor dem Traktor, rechtzeitiges Bremsen, Geschwindigkeitsanpassung
    • Traffic wird nicht nur direkt vor dem Traktor erkannt, sondern auch in Kurven (auf dem Kurs den der Traktor fährt).
    • (Nur im Combi-Modus) Auf dem Weg zum und vom Drescher werden Objekte mit Kollision (Bäume, andere Fahrzeuge) erkannt und umfahren (durch links/rechts ausschwenken).

  • Optionaler symmetrischer Bahnwechsel

    Für weitere Information bitte die FAQ lesen (“How to use the offset? — a.k.a. how do I let multiple vehicles drive a fieldwork course?”).

  • Optionale Einstellung im Drescher für die Abfahrer-Priorität

    Wahl zwischen Füllstand (der vollste Abfahrer wird gerufen) und Distanz (der naheste Abfahrer wird gerufen)

  • Einsteige-Knöpfe für globale Mitteilungen

    Steige direkt in das betroffene Fahrzeug ein (egal ob du in einem Traktor oder zu Fuß unterwegs bist) durch Benutzung des Mauscursors.

  • Support für folgende Mods hinzugefügt

    • AutoCombine (biedens)
    • DriveControl (upsidedown)
    • Overloader Script für Überlader (Satis)
